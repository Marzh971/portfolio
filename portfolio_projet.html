Lafleur

Compétences mobilisées :

  • 3. Développer la présence en ligne de l’organisation : Participer à l’évolution d’un site Web exploitant les données de l’organisation
  • 5. Mettre à disposition des utilisateurs un service informatique : Déployer un service

Description :

L'objectif du projet "Lafleur" consiste en la création d'un site web pour une entreprise du même nom. Lafleur est composé d'un menu de navigation permettant d'accéder aux différentes pages du site telles que la liste des produits par catégories. Ce site permet d'ajouter des produits dans un panier puis de passer une commande si l'utilisateur appartient à la base de donnée de l'entreprise. Le récaputilatif de cette commande sera ensuite enregistré dans la base de donnée.

Langage utilisé : PHP, HTML et CSS.

Projet PPE

Compétences mobilisées :

  • 4. Travailler en mode projet: Analyser les objectifs et les modalités d’organisation d’un projet / Planifier les activités / Évaluer les indicateurs de suivi d’un projet et analyser les écarts
  • 5. Mettre à disposition des utilisateurs un service informatique : Déployer un service

Description :

Ce projet s'appelle StatHammer et a été éffectué en groupe.
L'objectif de cette application est d'aider les joueurs de Warhammer 40k lors d'une partie en simulant des affrontements entre des unités (groupe de figurines ).
Ces unités appartiennent à des listes créées par l'utilisateur et sont stockées dans une base de données. L'application prend en compte le choix des armes , et des points de vie des figurines avant l'affrontement.

Langage utilisé : Java,Javascript,HTML et CSS.
